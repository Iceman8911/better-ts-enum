# Public Export Surface

This directory contains the public TypeScript modules exported from `better-ts-enum`. All modules listed here are part of the stable public API.

## Core Enum Types

- **`BasicEnum` / `BasicEnumBuilder`** ‚Äì Feature-rich enum with prefix/suffix, computed members, and advanced config. Import from `better-ts-enum/basic-enum`.
- **`MinimalEnum` / `MinimalEnumBuilder`** ‚Äì Lightweight enum with minimal API surface. Import from `better-ts-enum/minimal-enum`.

## Utilities

- **`arithmetic`** ‚Äì Numeric enum utilities (if available). Import from `better-ts-enum/arithmetic`.

See the individual module READMEs in the `enum-class/` directory for detailed documentation on each enum type.
